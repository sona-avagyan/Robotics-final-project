# Robot Path Planning API (Dijkstra Algorithm)

Дипломный/финальный проект по робототехнике, представляющий собой микросервис на базе **FastAPI** для планирования траектории мобильного робота на плоскости. В основе математического ядра лежит классический сеточный алгоритм **Дейкстры (Dijkstra)** с функцией инфляции препятствий (Obstacle Inflation) для безопасного обхода зон столкновения.

Проект разработан с учетом современных требований к архитектуре ПО в робототехнике: алгоритмическая часть полностью отделена от интерфейса взаимодействия (API).

## 🚀 Основные возможности

* **JSON Расчет пути (`/plan/text`)**: Возвращает точные координаты путевых точек (waypoints), общую длину маршрута и время работы алгоритма в секундах.
* **Статичная карта (`/plan/plot`)**: Генерирует и отправляет PNG-изображение со стартом, финишем, траекторией и визуализацией буферных зон безопасности вокруг препятствий.
* **Анимация поиска (`/plan/animate`)**: Создает GIF-анимацию, наглядно показывающую волновое расширение графа (исследованные узлы) и финальный маршрут с эффектом «замораживания» последнего кадра на 2 секунды.

## 🛠️ Алгоритмические особенности

1.  **Безопасный радиус (Obstacle Inflation)**: Алгоритм автоматически рассчитывает штрафную зону вокруг каждого препятствия по формуле $R_{safe} = R_{robot} + \frac{Res}{\sqrt{2}}$, что исключает "срезание углов" и столкновение робота с объектами при движении по диагонали.
2.  **Динамические границы**: Карта не привязана к координатам препятствий и может быть жестко ограничена размерами реального полигона тестирования.
